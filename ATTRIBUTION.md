## What We Did

The original OpenStax Organizational Behavior textbook was processed to:
- Extract key concepts, theories, and terms
- Identify relationships and connections between concepts
- Organize them into a structured knowledge graph format
- Create an interactive visualization using D3.js
- Provide multilingual labels (English, Chinese, Japanese)
